# UChat Server Documentation

This folder contains documentation for the UChat server application.

## Как читать Markdown документацию

### Visual Studio Code
1. **Preview режим:**
   - Откройте `.md` файл
   - Нажмите `Ctrl+Shift+V` (Windows/Linux) или `Cmd+Shift+V` (Mac)
   - Или нажмите на иконку "Open Preview" в правом верхнем углу редактора
   
2. **Side-by-side preview:**
   - Нажмите `Ctrl+K V` (Windows/Linux) или `Cmd+K V` (Mac)
   - Откроется preview рядом с исходным кодом

### Visual Studio (2019/2022)
- Markdown файлы автоматически открываются с preview
- Переключение между кодом и preview в верхней панели редактора

### JetBrains Rider / IntelliJ IDEA
- Откройте `.md` файл
- Справа появится вкладка "Preview" или кнопка разделения экрана
- Или нажмите `Ctrl+Shift+P` для toggle preview

### GitHub / GitLab / Bitbucket
- Markdown файлы автоматически рендерятся при просмотре в веб-интерфейсе

### Браузер
- Установите расширение для просмотра Markdown:
  - **Chrome/Edge:** Markdown Viewer, Markdown Preview Plus
  - **Firefox:** Markdown Viewer Webext
- Откройте `.md` файл в браузере

### Без IDE
- Любой текстовый редактор - структура понятна даже без рендеринга
- Онлайн редакторы: [dillinger.io](https://dillinger.io), [stackedit.io](https://stackedit.io)
